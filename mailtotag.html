Properties of the mailto HTML Tag

Following are the various properties of the mailto html tag.

  1. mailto: It is the main parameter that specifies the recipient's email address,
    including optional parameters such as CC (Carbon copy),
    BCC (Blind Carbon Copy), the subject and body of the message.
  2. Email address: It is the main parameter of the mailto tag that defines the address of the recipient mail field.
  3. Cc: It is an optional parameter of the mailto tag that
    holds the address of another mail and is received by the
    recipient as a carbon copy of the mail.
  4. Bcc: It is an optional parameter of the mailto tag
    that holds the specific address of another mail and
    is received by the recipient as a blind carbon copy of the mail.
  5. Subject: It is an optional parameter of the mailto tag.
    It is used to write the subject of the mail.
  6. Body: It is an optional parameter of the mailto tag.
    It is used to fill or write the content for mail.
  7. ?: It is an optional parameter of the mailto tag that
    holds the first parameter of the delimiter.
  8. @: It is also an optional parameter of the mailto tag
    that holds other delimiter parameters.
  9. Clickable text or image: It is the HTML hyperlink
    field that is displayed to the user on the browser.
