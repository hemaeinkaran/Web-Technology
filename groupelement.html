GroupingElements
Grouping Elements with < div > and < span > The < div > element is used to group block - level elements:
< div class=”footnotes” >
< h2 > Footnotes < /h2 >
< p > 1 The World Wide Web was invented by Tim Berners-Lee < /p >
< p > 2 The W3C is the World Wide Web Consortium which maintains many Web standards < /p >
< /div >

Footnotes

1 The World Wide Web was invented by Tim Berners-Lee

2 The W3C is the World Wide Web Consortium which maintains many Web standards


The < span > element, on the other hand, can be used to group inline elements only.
